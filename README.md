# Cheminformatics Internship Project

This repository contains the code and notebooks developed during my cheminformatics internship focused on molecular crystal structure analysis and classification.

## Overview

- **Foundational Data Processing:** Scripts for loading crystal data, calculating molecular descriptors, and preprocessing.
- **Conversion and Visualization Tools:** Utilities for converting formats (e.g., refcode to SMILES) and visualizing descriptor distributions.
- **Machine Learning Pipelines:** End-to-end machine learning workflows applying decision trees, logistic regression, random forests, and SVMs to classify crystal properties.

## Features

- Data cleaning, normalization, and feature selection.
- Model training with hyperparameter tuning and evaluation.
- Visualization of feature importance and model performance.
- Use of RDKit and CCDC Python APIs for chemical data handling.

## Technologies

- Python 3.x
- pandas, numpy
- scikit-learn
- matplotlib
- RDKit
- CCDC Python API
